# WP Tailwind Grid Block

A responsive, production-ready WordPress block powered by Tailwind CSS with scoped utility classes and flexible layout controls.

## Features

- Responsive grid with `grid-cols` and `gap` controls
- ARIA and semantic tag support
- Scoped Tailwind classes using `tgb:` prefix
- Editor styles for intuitive block selection
- Gutenberg-compatible with inner blocks

## Installation

1. Clone this repo into your WordPress `wp-content/plugins` directory
2. Run `npm install && npm run build`
3. Activate the plugin in your WordPress admin
